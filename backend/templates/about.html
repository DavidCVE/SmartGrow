Sistem Automat de Monitorizare Inteligentă a Serei

Autor Proiect de Diplomă: Cioveie David-Stefan
An Universitar: 2026
Instituție: Facultatea de Automatică și Calculatoare
Arhitectură Hardware

Stratul hardware IoT este responsabil pentru colectarea parametrilor de mediu și controlul fizic al sistemului de irigare. Nodurile comunică prin Wi-Fi direct cu baza de date centralizată în timp real.

  • Microcontroler ESP32

    Nodul IoT central. Se conectează la rețeaua locală Wi-Fi și acționează ca o punte de calcul între senzorii fizici și infrastructura cloud Firebase.

  • Senzor DHT22

    Furnizează citiri digitale de înaltă precizie atât pentru temperatura ambientală, cât și pentru umiditatea aerului din interiorul serei.

  • Senzor Capacitiv pentru Umiditatea Solului

    Măsoară conținutul volumetric de apă din sol. Designul său capacitiv previne problemele de coroziune întâlnite frecvent la senzorii rezistivi alternativi.

  • Modul Releu 5V

    Acționează ca un comutator electronic, separând în siguranță logica de joasă tensiune a ESP32 de cerințele de curent mai mari ale pompei de apă.

  • Pompă de Apă Submersibilă

    Execută comenzile de irigație automate, gestionate de modelul de Inteligență Artificială (Machine Learning), sau manuale, pentru a menține condițiile ideale de umiditate a solului pentru culturi.

Arhitectură Software

Stratul software utilizează o arhitectură cloud decuplată pentru a asigura o disponibilitate ridicată, autentificare sigură a utilizatorilor și panouri de administrare responsive, care folosesc un model de Machine Learning pentru îngrijirea inteligentă a plantelor.

  • Backend Python Flask

    Un framework de aplicații web WSGI ușor și robust, care servește drept backend REST API. Acesta gestionează rutele securizate, randează șabloane dinamice, asigură procesarea datelor și integrează modelul AI.

  • Firebase Realtime Database

    O bază de date NoSQL găzduită în cloud. Datele sunt sincronizate pe toți clienții în timp real și rămân disponibile atunci când aplicația este offline.

  • Firebase Authentication

    Oferă servicii backend robuste și biblioteci UI gata de utilizare pentru autentificarea utilizatorilor, asigurându-se că panourile senzorilor și controalele sunt sigure.

  • Bootstrap 5 & JavaScript

    Frontend-ul modern este construit folosind Bootstrap 5 pentru un sistem de grid responsive, orientat spre mobile. JavaScript simplu (Vanilla JS) este combinat cu Fetch API pentru actualizări asincrone pe partea de client.
